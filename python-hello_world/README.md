# Holberton School - Higher Level Programming

This repository contains all the projects for the Higher Level Programming track at Holberton School.
The projects are written in Python and focus on learning the foundations of higher-level languages, scripting, algorithms, and software engineering concepts.

## Repository Structure
- python-hello_world: Introduction to Python programming.

## Requirements
- Python 3.8 (Ubuntu 20.04 LTS)
- pycodestyle 2.7
- All scripts must be executable

## Task 0
- **File:** 2-print.py
- **Description:** Prints exactly "Programming is like building a multilingual puzzle" followed by a new line.
