# Holberton School - Higher Level Programming

This project is part of the Higher Level Programming track at Holberton School.
It introduces Python programming and focuses on the foundations of higher-level languages, scripting, and software engineering concepts.

## Repository Structure
- python-hello_world: Introduction to Python programming

## Requirements
- Python 3.8 (Ubuntu 20.04 LTS)
- pycodestyle 2.7
- All scripts must be executable
